# Todo List

Simple Todo List with React and TypeScript

![a-screenshot-of-todolist](screenshots/todolist.png)

## Features

- Create a new item
- Delete an item
- Drag and drop an item by using react-beautiful-dnd (https://www.npmjs.com/package/react-beautiful-dnd)
